# Agent Skills for Beginners
- Learn how to Create and User Agent Skills
- Learn How to Use Agent Skills with AI Agents
- Learn how to create Agent Skills with `bash`
- Learn how to create Agent Skills with `python script`
## How to Run
- Clone this repo
- Go to project root
- Open terminal
- Launch AI Agent such as Antigravity by executing `agy` command
- Use following prompts
1. **gender-finder Skill** : "What is the gender of name `Smith`" 
2. **car-sales-finder Skill** : "What is Car sales for `April 2026`"
